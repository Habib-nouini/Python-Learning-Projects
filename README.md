# My Python Learning Journey

مرحباً بك في مستودعي الشخصي لتعلم لغة بايثون. هنا أوثق رحلتي في تعلم البرمجة من خلال حل المسائل وبناء مشاريع صغيرة.

## المشاريع
- **Seconds Converter**: برنامج لتحويل الثواني إلى صيغة الوقت.
- **Area Calculator**: برنامج لحساب مساحة المستطيل وتكلفتها بناءً على سعر المتر.
